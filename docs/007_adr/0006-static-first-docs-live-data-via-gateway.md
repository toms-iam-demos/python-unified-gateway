# ADR-0006: Documentation Is Static-First; Live Data Access Requires Gateway APIs

## Intent
Define safe documentation and interactivity practices.

## Status
Accepted

## Decision
Documentation is static-first.
Live data access requires gateway-mediated APIs.

## Rationale
- Browsers are untrusted
- Live data requires policy enforcement

## Scope
Applies to documentation and interactive examples.

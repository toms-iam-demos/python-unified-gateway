# ADR-0008: Pro Notebooks Are Authoring Tools; Published Artifacts Are Static or Browser-Safe

## Intent
Separate authoring environments from published runtime behavior.

## Status
Accepted

## Decision
Full-capability notebooks are authoring-only.
Published artifacts must be static or browser-executed.

## Rationale
- Security
- Reproducibility
- Accessibility

## Scope
Applies to notebooks and documentation publishing.
